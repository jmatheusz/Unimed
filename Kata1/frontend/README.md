# Front-End de Triagem Médica (Kata 1)

Este é o front-end da aplicação de Triagem Médica, desenvolvido com tecnologias modernas para garantir uma interface rápida, responsiva e intuitiva.

## 🛠 Tecnologias Utilizadas

*   **React:** Biblioteca principal para construção da interface.
*   **TypeScript:** Garante segurança de tipos e melhor manutenção do código.
*   **Vite:** Ferramenta de build extremamente rápida.
*   **CSS Moderno:** Estilização focada em usabilidade e estética premium.

## 🚀 Como Executar

1.  Certifique-se de estar na pasta `Kata1/frontend`.
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
4.  Acesse a aplicação no navegador através do endereço exibido no terminal (geralmente `http://localhost:5173`).

---

## 📖 Notas de Desenvolvimento

Este front-end consome a API desenvolvida em .NET Core. Ele permite que os profissionais de saúde visualizem a fila de pacientes ordenada por prioridade (seguindo as regras de idade e urgência clínica).
