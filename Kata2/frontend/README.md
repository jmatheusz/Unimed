# Painel de Tarefas - Front-End (Kata 2)

Interface moderna para o gerenciamento de tarefas (TaskFlow), focada em produtividade e organização.

## 🛠 Tecnologias Utilizadas

*   **React + TypeScript:** Base da aplicação para uma experiência de usuário reativa e segura.
*   **Vite:** Tooling de próxima geração para desenvolvimento rápido.
*   **Arquitetura Baseada em Componentes:** Facilita a manutenção e escalabilidade da UI.

## 🚀 Como Executar

1.  Certifique-se de estar na pasta `Kata2/frontend`.
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
4.  Acesse `http://localhost:5173` no seu navegador.

---

## 💡 Funcionalidades

*   **Criação de Tarefas:** Adicione novas obrigações rapidamente.
*   **Filtros Inteligentes:** Visualize apenas o que importa (Pendentes, Concluídas ou Todas).
*   **Gestão de Status:** Marque tarefas como concluídas com um clique.
*   **Design Responsivo:** Funciona perfeitamente em desktops, tablets e smartphones.
